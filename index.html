Les types de base


Il existe la base 2 ( binaire )

Représentation des nombres entiers

En décimal 123 = 1*102 + 2*101 + 3*100
= 123(10)

Représentation de nombre binaire

en binaire (base2), 1011 = 1*23 0*22 1*21 1*20 = 11(10)

Représentation de nombre hexadécimal

en hexadécimal (base16), "fafa" = 15*163 10*162 15*161 10*160 = 64250(10)

Autres exemples :

0111 0000 0001 1111 (2) = 701F (16)


Nous réunissons par groupe de 4 les symboles et les convertissons en base 16: 0111 = 7 ==> 0000 = 0 ==> 0001 = 1 ==> 1111 = F

Il existe aussi des nombres binaire négatifs ou des nombres binaire a virgules


Le binaire des négatifs

Les nombres négatifs sont codés de façon que les opérations classiques puisent être effectuées normalement.
Autrement dit, la somme de 2 et (-2) doit bien donner 0.


Les nombres négatifs sont codés selon leur complément de deux

Exemple :

Prenons 21, sa valeur binaire est 010101.
On inverse les bits cela nous donne : 101010 ensuite, nous ajoutons 1, cela nous donne 101011. Et voila, -21
le bit tout a droite est celui des signes, 0 = + ; 1 = - (pour faire simple)
donc le bit du poid 32 est le bit des signes. Si l'on reinverse les bits et ajoute 1, nous retrouvons 21.

Et voila, le cour est teminé !
